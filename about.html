Project Description

This is a web-based Turing Machine Simulator built using HTML, CSS, and JavaScript. It visually demonstrates how a Turing Machine processes input step-by-step.

Features

  • Step-by-step execution
  • Automatic simulation
  • Execution tracking table
  • Speed control
  • Modern UI
