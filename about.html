Project Description

This project is a web-based Turing Machine Simulator built using HTML, CSS, and JavaScript. It visually demonstrates how a Turing Machine processes input step-by-step.

Features

  • Manual (Step-by-step) execution
  • Automatic simulation mode
  • Execution tracking table
  • Speed control slider
  • User-defined transition rules (dropdown)
  • Multi-page UI (Home, About, Theory)
